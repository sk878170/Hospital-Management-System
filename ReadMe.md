# Hospital Management System

A web-based Hospital Management System developed as a **Third-Year B.E. Project** at **ISBM College of Engineering**. This project is designed to simplify hospital operations by providing a platform for **patients, doctors, and administrators** to interact efficiently. It allows users to book appointments and consult doctors online through an easy-to-use interface [file:1].

## Project Overview

The Hospital Management System helps streamline basic hospital activities and improves communication between patients and medical staff. The project focuses on creating a simple and effective website interface for managing appointments, doctor consultations, and user interactions in a digital environment [file:1].

## Features

- Patient registration and login
- Doctor registration and login
- Admin panel for management
- Online appointment booking
- Doctor consultation interface
- User-friendly dashboard
- Basic hospital record handling

## Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **MySQL**

## Modules

### Patient Module
Patients can register, log in, book appointments, and consult doctors through the system.

### Doctor Module
Doctors can log in, view appointment requests, and manage patient consultations.

### Admin Module
The admin manages overall system activities, users, and hospital-related data.

## Objectives

- To digitize basic hospital management tasks
- To reduce manual work in appointment handling
- To improve communication between patients and doctors
- To provide a simple and accessible healthcare management platform

## How to Run the Project

1. Download or clone this repository.
2. Open the project folder in your code editor.
3. Set up the MySQL database.
4. Import the required database tables.
5. Open the project in your browser using Live Server or a local server.
6. Start using the system.
